#Finance manager
Its finance company software. You can manage customer's loan, EMI reminder, Out-standing cash report. Also you can manage profit from your emi interest.

You can contact me for any customisation require. Just drop email in "lalitpatel.lp@gmail.com" 


##Instalation
1. Clone this repository or you can download and unzip file. Put it on your document root directory.
2. Create new database and add database details in app/config/database.php file
3. Open website in your favourite browser. http://localhost/finance-manager
<br />It will automatically create all require tables in database.
4. Insert admin user (password: admin@123)
